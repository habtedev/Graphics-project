# 🎨 Graphics Project (C++ OpenGL / Graphics Demos)

![C++](https://img.shields.io/badge/language-C%2B%2B-blue.svg)
![Status](https://img.shields.io/badge/status-active-success.svg)
![License](https://img.shields.io/badge/license-MIT-blue.svg)

Compact C++ graphics demos: `flag.cpp` and `logo.cpp`, plus rendered outputs in `image/`.

--


**Quickstart**

- Clone the repo:

```bash
git clone https://github.com/habtedev/Graphics-project.git
cd Graphics-project
```

- Build (single binary for quick testing):

```bash
g++ flag.cpp logo.cpp -o graphics_demo
```

- Run:

```bash
./graphics_demo    # Linux / macOS
.\graphics_demo.exe  # Windows
```

Or build per-example:

```bash
g++ flag.cpp -o flag_demo
g++ logo.cpp -o logo_demo
```

**What you'll find**

- `flag.cpp` — draws a sample flag.
- `logo.cpp` — draws a simple logo.
- `image/` — rendered screenshots (5 images).
- `LICENSE` — project license.

**Requirements**

- C++ compiler with C++11 or newer (`g++`, `clang++`).
- On Windows: MinGW-w64 or WSL recommended.
